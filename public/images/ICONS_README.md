# PWA Icons Required

يجب إضافة أيقونات PWA في هذا المجلد بالأحجام التالية:

## Required Icon Sizes:
- icon-72x72.png
- icon-96x96.png
- icon-128x128.png
- icon-144x144.png
- icon-152x152.png
- icon-192x192.png
- icon-384x384.png
- icon-512x512.png

## How to Generate Icons:

يمكنك استخدام أي من هذه الأدوات لإنشاء الأيقونات:

1. **PWA Asset Generator** (Recommended)
   - Visit: https://www.pwabuilder  .com/imageGenerator
   - Upload your logo (recommended: 512x512px or higher)
   - Download generated icons

2. **RealFaviconGenerator**
   - Visit: https://realfavicongenerator.net/
   - Upload your logo
   - Download the generated package

3. **Manual Creation**
   - Use any image editor (Photoshop, GIMP, Figma, etc.)
   - Create a square logo
   - Export in the sizes listed above

## Icon Guidelines:

- **Format**: PNG with transparency
- **Shape**: Square (1:1 ratio)
- **Background**: Transparent or solid color (#b65f7a recommended)
- **Safe Zone**: Keep important content within 80% of the canvas
- **Purpose**: Icons should be recognizable at all sizes

## Brand Colors:
- Primary: #b65f7a
- Dark: #8b4558
- Light: #e8a1b7

## Quick Start:

If you need placeholder icons temporarily, you can:
1. Create a simple 512x512px PNG with your brand color
2. Use an online tool to resize it to all required sizes
3. Replace with professional icons later
